# xsbt-web-plugin

xsbt-web-plugin is tool for building J2EE Web applications in Scala.

## Features

* Build a project on Java servlets
* Package a project as a *.war* file
* Run a project under Jetty or Tomcat
* Deploy a project to Heroku or AWS
